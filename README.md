# Kassabuch-App (Android)

Eine einfache Android-App zur Erfassung von Einnahmen und Ausgaben mit Monatsübersicht, Budget und Auszahlungsterminen.

## Funktionen (MVP)
- Einnahmen erfassen und verwalten
- Ausgaben erfassen (fix & variabel)
- Monatsübersicht mit Saldo
- Auszahlungstermine anzeigen und bearbeiten
- Monatsbudget setzen und Warnhinweise bei Überschreitung
- Kategorien für Einnahmen und Ausgaben verwalten
- Historie für fixe Werte (Änderungen gelten ab Datum)

## Technik
- Android (Jetpack Compose)
- Lokale Speicherung (Room)
- Sprache: Deutsch
- Währung: EUR
